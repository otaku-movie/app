# 変更履歴

言語： [English](CHANGELOG.md) | [简体中文](CHANGELOG.zh-CN.md) | **日本語**

本ファイルは シネコ（otaku_movie）アプリの各バージョンの主な変更点を記録します。

## [1.0.1+2] - 2026-06-12

### 追加
- 特典導線の計測を強化：「受け取り可能な劇場を見る」、特典シェア、特典画像プレビュー、特典フィードバック送信、劇場の絞り込み・劇場タップなどの主要な行動を計測し、特典のコンバージョン導線を追跡しやすくしました。
- 注文詳細ページの特典フィードバック送信の計測を追加しました。
- API エラー計測で送信される情報を改善しました。
- 映画リストのデータ項目を拡張し、映画詳細・上映スケジュール・検索ページ・劇場リストでより充実した映画・劇場情報を表示するようにしました。

### 改善
- 多言語テキストの更新：パスワードルールの案内を「8〜32 文字、英字と数字を含む、記号使用可」に変更し、パスワード強度「弱/中/高」の表記を追加、著作権およびデータ免責事項の連絡先メールアドレスを更新しました。
- 特典フェーズの状態表記を「以前」から「開始前」に変更し、意味を明確にしました。
- 辞書の多言語表示をより正確にしました。
- グローバルな AppBar を白色に統一し、Material 3 でスクロール時にタイトルバーが灰色に着色される問題を修正しました。
- 登録ページの入力欄プレフィックスアイコンのサイズと配置を統一し、認証コード「送信」ボタンのアイコンとテキストを拡大しました。
- 個人情報ページの項目フォントを拡大し、可読性を向上しました。

### 修正
- 登録ページのパスワード入力欄の右側のレイアウトはみ出しを修正し、長いエラーメッセージが正しく折り返されるようにしました。

### その他
- Android のビルド設定を調整し、ローカルデバッグとリリースビルドの境界を明確にしました。
- バージョンを `1.0.1+2` に更新しました。
