# 更新日志

语言： [English](CHANGELOG.md) | **简体中文** | [日本語](CHANGELOG.ja.md)

本文件记录 シネコ（otaku_movie）App 各版本的主要变更。

## [1.0.1+2] - 2026-06-12

### 新增
- 完善特典与业务链路埋点：覆盖「查看可领影院」、特典分享、特典图片预览、特典反馈提交、可领影院筛选与影院点击等关键行为，便于追踪特典转化链路。
- 补充订单详情页的特典反馈提交埋点。
- 优化 API 错误埋点上报信息。
- 扩展电影列表数据字段，电影详情、场次列表、搜索页与影院列表展示更完整的影片与影院信息。

### 优化
- 多语言文案更新：调整密码规则提示为「8-32 位，包含字母和数字，可使用特殊字符」，新增密码强度「弱/中/高」文案，更新版权与数据免责声明联系邮箱。
- 将特典阶段状态文案「之前」调整为「未开始」，表意更清晰。
- 字典在多语言下的展示更准确。
- 全局 AppBar 统一为白色，修复 Material 3 下页面滚动时标题栏被着色变灰的问题。
- 统一注册页输入框前缀图标的尺寸与对齐，放大验证码「发送」按钮的图标与文字。
- 放大个人信息页字段字体，提升可读性。

### 修复
- 修复注册页密码输入框右侧布局溢出，长错误提示可正常换行。

### 其他
- 调整 Android 构建配置，保持本地调试与发布构建的边界清晰。
- 版本号更新至 `1.0.1+2`。
